## 核心理论框架

### MECE 学习力四维度

| 字母 | 维度 | 说明 |
|---|---|---|
| **M** | 学习动力 (Motivation) | 内驱力 / 目标感 / 抗压韧性 |
| **E** | 学习执行力 (Execution) | 时间管理 / 任务完成率 / 自我监控 |
| **C** | 学习能力 (Capability) | 专注力 / 记忆力 / 推理理解 / 学习方式 |
| **E** | 学习环境 (Environment) | 家庭支持 / 同伴环境 / 学习空间 |

> ⚠️ MECE 在本系统中是「提分密码」4维度，**不是** 咨询术语"Mutually Exclusive"。

## 测评工具体系

1. **DISC** — 行为风格测评(老虎/孔雀/考拉/猫头鹰)
2. **MECE 综合报告** — 学习力 4 维度综合测评
3. **MBTI** — 人格类型
4. **霍兰德** — 职业兴趣
5. **多元智能** — 8 项智能分布
6. **学习方式偏好** — 视觉/听觉/动觉
7. **SCL-90** — 心理健康筛查
8. **职业锚** — 职业发展方向
9. **JUMEQ** — 升学密码测评
10. **CAMIQ** — 职业密码测评

## 与主仓库的联系

- 测评结果 → 写入 `pipeline-data/quadruple-actions.json` 四元组
- MECE 诊断 → 驱动 `agent-skills/student/` 的学习路径推荐
- 七阶掌握度(不会→模糊→清晰→框架→运用→熟练→创新)基于 MECE 诊断动态调整
